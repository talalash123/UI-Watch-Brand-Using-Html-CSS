# UI-Watch-Brand-Using-Html-CSS
A premium, high-performance E-commerce UI designed for a luxury Swiss watch brand. This project focuses on high-end aesthetics, smooth micro-interactions, and a seamless user journey.


##  Preview




---

##  Features

- **Modern UI/UX:** Built with a "Black & Gold" luxury theme using Cinzel and Inter typography.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.
- **Interactive Cart:** Real-time side-cart with subtotal calculation.
- **Smart Filtering:** Instant category-based filtering (Professional vs. Classic).
- **Smooth Animations:** Integrated scroll-reveal and hover-zoom effects for a premium feel.

---

##  Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Grid)
- **Scripting:** Pure JavaScript (ES6+)
- **Fonts:** Google Fonts (Cinzel & Inter)
- **Images:** High-Resolution Unsplash Assets
